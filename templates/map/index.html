    About This Project

    Code Green started in 2021 as my first green space analysis of Wilmington, NC — a manual satellite image study I built from scratch. This version expands that foundation into a live, multi-city tracker using real NASA/USGS Landsat 8 and Sentinel-2 data, rebuilt with Claude as a YSA Global STEM Ambassadors research project.

    The research tests whether COVID-19 lockdowns paused construction in 2020, creating a post-pandemic development surge that accelerated green space loss — comparing Wilmington, NC with Chisinau and Lozova, Moldova.

    The Research Hypothesis
    COVID-19 paused construction in 2020. When restrictions lifted, an accelerated development surge tested whether green infrastructure in both the US and Eastern Europe faced the same pressure.
